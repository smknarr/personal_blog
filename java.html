Project Title

Online Patient Records Management Portal

Abstract

Huge amounts of patient records data is generated each and every single day and storage and accessibility of these records is a huge concern. This project aims to provide a viable solution for the same.

Features

  • Easy and user-friendly ui to add patient details
  • Can store huge number of records
  • Easy deletion of records based on patient name or patient ID
  • Interactive way to access details by either name, ID, area of the hospital or disease
